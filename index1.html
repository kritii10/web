Welcome!

This tool helps detect thread deadlocks and monitor CPU burnout in real time using Python and DTrace integration.

About the Tool

- Detects deadlocks by monitoring thread locking.
- Monitors CPU usage to detect system burnout.
- Sends alert emails automatically for high CPU conditions.
- Integrates with DTrace for deep system-level tracing.

How It Works

The Python script monitors threads and CPU usage. If CPU usage remains above 80% for 5 cycles, it triggers an email alert. DTrace can be used to monitor thread locks and detect deadlocks on a deeper level.
